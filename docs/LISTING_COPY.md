# Listing Copy

## Short Description

MCP server for reading, editing, and publishing Google Forms with the official Google APIs.

## Longer Description

Google Forms MCP gives AI agents and developers a typed, reliable interface for working with Google Forms over MCP. It can inspect a form, list normalized items, update form copy, add and edit questions, move or delete items, attach images, read responses, and manage publish state. It uses the official Google Forms API and optionally Google Drive permissions when responder access changes require it. This is useful for intake automation, client onboarding forms, survey maintenance, and any workflow where agents need to safely modify real Google Forms instead of relying on brittle browser automation.
